# Michael Portfolio

Website portofolio pribadi untuk menampilkan proyek web, pengalaman QA, dan kontak profesional.

## Fitur

- Layout responsif untuk desktop dan mobile
- Navigation sticky dengan smooth scroll antar bagian
- Kartu proyek dengan efek reveal saat di-scroll
- Stuktur sederhana dan mudah dikembangkan

## Teknologi

- HTML
- CSS
- JavaScript

## Struktur Folder

- `index.html` - halaman utama portofolio
- `css/style.css` - gaya tampilan
- `js/main.js` - interaksi JavaScript (smooth scroll, efek scroll)
- `assets/` - tempat menyimpan gambar atau file tambahan

## Cara Menjalankan

1. Buka `index.html` langsung di browser
2. Atau gunakan extension Live Server di VS Code
