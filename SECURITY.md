# Security Policy

## Supported Versions

This project is maintained as a personal portfolio. The latest version available on the `main` branch is considered the supported version.

| Version | Supported |
|----------|-----------|
| Latest   | ✅ |
| Older    | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

You can contact me through:

- Email: your-email@example.com
- GitHub Issues (for non-sensitive issues)

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested mitigation (if available)

I will review reported issues as soon as possible and work on an appropriate fix.

Thank you for helping improve the security of this project.
